# Smart Notes App 

A full-stack notes application with tag-based filtering and search.

##  Features
- Create and delete notes
- Add tags to notes
- Search notes by title/content
- Filter notes by tags

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Storage: JSON file

##  Setup

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm start
